How to Play Sudoku

Objective

Fill the 9Ã—9 grid with digits so that each column, each row, and each of the nine 3Ã—3 subgrids (also called "boxes" or "regions") contains all of the digits from 1 to 9.

Rules

  1. Each row must contain the numbers 1-9 exactly once.
  2. Each column must contain the numbers 1-9 exactly once.
  3. Each 3Ã—3 box must contain the numbers 1-9 exactly once.
  4. You cannot change the numbers that are already filled in (these are shown in a different color).

Tips

  • Start by looking for rows, columns, or boxes that have many numbers already filled in.
  • Look for numbers that can only go in one place in a row, column, or box.
  • Use the "Hint" button if you get stuck - it will show you the next correct number.
  • Take your time and think logically - there's only one correct solution!
